# Nosso Universo 🌌

Um mural interativo e infinito de memórias estelares. Cada memória (foto + data + descrição) torna-se um planeta em órbita no seu próprio universo.

## 🚀 Tecnologias

- **Frontend**: HTML5, CSS3, JavaScript (Canvas API)
- **Backend**: Supabase (Database & Storage)
- **Design**: Estética espacial com animações interativas e efeitos sonoros narrativos.

## ⚙️ Configuração

### Supabase Setup

1. Crie um projeto no **Supabase**.
2. No **SQL Editor**, execute o script de criação da tabela `memories`.
3. No **Storage**, crie um bucket público chamado `memories`.
4. Configure as chaves `SUPABASE_URL` e `SUPABASE_KEY` no arquivo `app.js`.

### Rodando Localmente

Basta abrir o arquivo `index.html` em qualquer navegador moderno.
